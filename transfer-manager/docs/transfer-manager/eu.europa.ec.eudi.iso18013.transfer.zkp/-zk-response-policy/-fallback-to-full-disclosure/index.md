# FallbackToFullDisclosure

[release]\
data object [FallbackToFullDisclosure](index.md) : [ZkResponsePolicy](../index.md)

Fall back to full document disclosure if ZK proof generation fails. This is the current default for backwards compatibility and will be changed to [Strict](../-strict/index.md) in a future release.
